# AI Assistant (В разработке 🚧)

Android-приложение для голосового ИИ-ассистента. Проект находится на стадии активной разработки (Work in Progress).

## Что уже реализовано

Интерфейс и базовая архитектура приложения:
* **UI:** Сверстаны основные экраны на Jetpack Compose (главный экран, история чатов, экран переписки, настройки).
* **Навигация:** Настроены переходы между экранами.
* **Архитектура:** Заложен фундамент проекта (разделение на слои, DI через Koin).
* **Сеть:** Подготовлен базовый клиент на Ktor для будущих сетевых запросов.
* **UI-компоненты:** Готовы элементы управления (кнопка записи, слайдеры настроек, карточки сообщений).

## Что будет сделано (В планах)

Основной функционал ассистента находится в разработке. В ближайшее время планируется реализовать:

* **Foreground Service:** Для поддержания работы ассистента в фоновом режиме.
* **Распознавание триггер-слов:** Интеграция **Porcupine** для локальной активации ассистента голосом (wake word).
* **Работа со звуком:** Запись аудио и его транскрибация (Speech-to-Text) с использованием нативных библиотек Android.
* **LLM API:** Интеграция с языковой моделью через Ktor для генерации осмысленных ответов.

## Стек технологий

* Kotlin
* Jetpack Compose
* Ktor Client
* Koin
